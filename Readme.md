building a console based 3MTT calculator

This project is a console-based Smart Calculator built with Kotlin.

The purpose of This project console-based Smart Calculator built with Kotlin is to help officials of the 3MTT computer distribution project so that users or officials can input arithmetic numbers and the calculator uses the operator selected by the user to complete the calculation.
